### Desarrollo de la práctica 'Introducción a Nextra'

En esta práctica se ha desarrollado una aplicación web con Nextra, un generador de sitios web estáticos basado en Next.js. 
Para ello, se ha seguido la documentación oficial de Nextra y se ha personalizado el contenido de la página principal.

### Contenido de la página principal

La página principal de la aplicación web contiene una breve descripción de la práctica y un enlace a la documentación oficial de Nextra.
 Además, se ha añadido un enlace a la página de GitHub del proyecto.

### Despliegue de la aplicación web

La aplicación web se ha desplegado en Vercel, una plataforma de computación en la nube que permite desplegar aplicaciones web de forma sencilla y rápida.

### Conclusiones

En esta práctica se ha aprendido a utilizar Nextra para generar sitios web estáticos con Next.js. 
Nextra es una herramienta muy potente y fácil de usar que permite crear sitios web estáticos de forma rápida y sencilla.
